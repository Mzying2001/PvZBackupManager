## 关于软件

本软件是植物大战僵尸存档管理器，用于备份/恢复植物大战僵尸的游戏存档

## 支持版本

软件支持原版（包括年度版）、Steam版、Zoo日文版

对于年度加强版（非精简版），在恢复存档前必须先关闭游戏，否则会报错
