## 关于软件

本软件是植物大战僵尸存档管理器，用于备份/恢复植物大战僵尸的游戏存档。

## 支持版本

|  原版  | 年度版 | Steam版 | Zoo日文版 | 年度加强版 |
| :----: | :----: | :-----: | :------: | --------: |
| ✅支持 | ✅支持 | ✅支持 |  ✅支持  | ❌半支持 |

以上为对不同版本游戏的支持情况，其他未测试版本只要存档路径与上面版本相同且运行时不会占用存档文件则理论上是支持的。对于年度加强版，由于该版本在游戏运行时会占用存档文件导致无法修改存档，需要先关闭游戏后才可以恢复存档。
