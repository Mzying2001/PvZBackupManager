## 关于软件

本软件是植物大战僵尸存档管理器，用于备份/恢复植物大战僵尸的游戏存档

## 支持版本

软件支持原版（包括年度版）、Steam版、Zoo日文版等

对于一些版本（如年度加强版），若恢复存档时报错，可关闭游戏后重试，这是因为在Windows系统下无法修改被其他程序占用的文件，不是bug
